# README.md

## DigitalOcean Static Site Starter

This repository contains a lightweight, SEO-ready static website intended for deployment on DigitalOcean App Platform (Static Site).

### Structure
- `index.html` – main site file
- `assets/css/styles.css` – styles
- `assets/js/main.js` – small enhancements
- `robots.txt` – crawler rules
- `sitemap.xml` – sitemap

### Deploy
1. Push to GitHub
2. DigitalOcean App Platform → Create App → GitHub repo
3. Choose Static Site
4. Build command: *(blank)*
5. Output directory: `/`
